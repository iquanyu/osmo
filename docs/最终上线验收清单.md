# 最终上线验收清单

> 使用方式：发布前按顺序逐项勾选；有一项未通过，就先不要上线。

## 1. 权限与入口

- [ ] `admin / player` 角色判定正确
- [ ] 普通玩家无法进入任何后台路由
- [ ] 禁用登录用户无法继续访问受保护页面
- [ ] 后台页页头、侧边栏、返回入口一致

## 2. 后台关键链路

- [ ] 运营总览可正常访问
- [ ] 教程管理：列表、筛选、新增、编辑、删除可用
- [ ] 社区管理：列表、详情、置顶、删除、官方回复可用
- [ ] 投稿审核：列表、详情、通过、驳回可用
- [ ] 用户管理：角色切换、重置密码、禁用/恢复登录、导出可用
- [ ] 日志管理：筛选、详情、导出可用

## 3. 前台关键链路

- [ ] 首页入口分流清晰
- [ ] 教程详情页“内容 + 工具”闭环可用
- [ ] ND 计算器可用
- [ ] 工具页可用
- [ ] 社区可浏览、可发帖、可回复
- [ ] 投稿台可创建、编辑、提交、删除

## 4. 数据与发布

- [ ] 角色迁移已执行
- [ ] 登录禁用迁移已执行
- [ ] 日志表迁移已执行
- [ ] 生产环境构建可成功生成前端资源
- [ ] 部署后页面无 manifest 缺失

## 5. 测试与回归

- [ ] 关键后台功能测试通过
- [ ] 权限边界测试通过
- [ ] 危险操作与失败分支覆盖
- [ ] 构建通过

## 6. 监控与恢复

- [ ] 错误日志可查
- [ ] 关键后台操作有日志记录
- [ ] 可恢复演示数据或可回滚
- [ ] 导出、删除、驳回等操作有明确反馈

## 7. 结论

- [ ] 全部通过，允许上线
- [ ] 有未通过项，暂停上线
