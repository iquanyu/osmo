# 环境变量说明

这份文档只列和 Osmo 有关的关键环境变量。

---

## 1. 应用基础

| 变量 | 说明 |
|---|---|
| `APP_NAME` | 应用名称 |
| `APP_ENV` | 运行环境 |
| `APP_KEY` | 加密密钥 |
| `APP_DEBUG` | 调试模式 |
| `APP_URL` | 站点根 URL |

## 2. 数据库

| 变量 | 说明 |
|---|---|
| `DB_CONNECTION` | 数据库驱动 |
| `DB_HOST` | 数据库主机 |
| `DB_PORT` | 数据库端口 |
| `DB_DATABASE` | 库名 |
| `DB_USERNAME` | 用户名 |
| `DB_PASSWORD` | 密码 |

本地默认用 SQLite。

## 3. 会话与缓存

| 变量 | 说明 |
|---|---|
| `SESSION_DRIVER` | 会话驱动 |
| `SESSION_LIFETIME` | 会话时长 |
| `CACHE_STORE` | 缓存驱动 |

## 4. 文件存储

| 变量 | 说明 |
|---|---|
| `FILESYSTEM_DISK` | 默认磁盘 |
| `AWS_ACCESS_KEY_ID` | S3 访问密钥 |
| `AWS_SECRET_ACCESS_KEY` | S3 密钥 |
| `AWS_DEFAULT_REGION` | 区域 |
| `AWS_BUCKET` | 存储桶 |

封面图上传默认写到 `tutorials/` 和 `submissions/`。

## 5. 邮件

| 变量 | 说明 |
|---|---|
| `MAIL_MAILER` | 邮件驱动 |
| `MAIL_HOST` | SMTP 主机 |
| `MAIL_PORT` | SMTP 端口 |
| `MAIL_USERNAME` | 用户名 |
| `MAIL_PASSWORD` | 密码 |
| `MAIL_FROM_ADDRESS` | 发件地址 |

## 6. 前端

| 变量 | 说明 |
|---|---|
| `VITE_APP_NAME` | 浏览器标题前缀 |

## 7. 场景助手

当前场景助手使用本地预设规则；后续如果接入外部模型，再补相应变量。

## 8. 安全提示

- 不要提交 `.env`
- 生产环境关闭 `APP_DEBUG`
- 定期轮换数据库和对象存储凭证
- 管理员账号使用强密码
